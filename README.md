## 🎯 About

Modern, responsive portfolio with smooth animations and a premium feel. Built to impress potential clients and employers.

## ✨ Features

- 🎨 Stunning visual design
- 🌙 Dark/Light theme toggle
- 📱 Fully responsive
- ⚡ Blazing fast performance
- 🎭 Smooth animations
- 📊 Project showcase
- 📧 Contact form

## 🛠️ Tech Stack

- Next.js
- React
- Tailwind CSS
- Framer Motion
- Vercel (Hosting)

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/4harmony55-glitch/paul-portfolio-v2.git

# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```
